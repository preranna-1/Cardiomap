# Healthcare Management System

A web-based healthcare management system developed using Node.js, Express, MySQL, HTML, CSS, and JavaScript.

## Features
- Patient registration
- Doctor management
- Appointment scheduling
- Database integration
- CRUD operations

## Technologies Used
- Node.js
- Express.js
- MySQL
- HTML/CSS
- JavaScript

## How to Run

1. Clone the repository
2. Run npm install
3. Import demos.sql into MySQL
4. Configure db.js
5. Run node server.js
